## Update 工作流

> 更新现有 Wiki 页面

### 触发条件

- 源文档更新
- 发现错误需要修正
- 添加新链接

### 流程步骤

```
Step 1: 确定需要更新的页面
  → 读取目标文件
  → 理解现有内容

Step 2: 执行更新
  → 修改内容
  → 添加/修复链接
  → 更新updated日期

Step 3: 验证
  → 检查链接有效性
  → 检查格式一致性

Step 4: 记录
  → 更新log.md
```

### Update Prompt 模板

```markdown
## Update任务

### 输入
- 目标文件: wiki/concepts/[领域]/[概念名].md
- 更新内容: [具体修改说明]

### 执行
1. 读取目标文件
2. 执行修改
3. 检查链接
4. 更新updated日期
5. 记录到log.md
```
