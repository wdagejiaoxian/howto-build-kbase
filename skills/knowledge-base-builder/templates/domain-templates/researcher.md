# 学术研究者模板

## 适用身份
研究生、教授、研究员

## 预设分类

```
原始资料/
├── 论文/              # 学术论文、期刊文章
├── 实验方法/          # 实验设计、数据采集
├── 数据集/            # 公开数据集、自采数据
├── 理论/              # 理论框架、数学模型
├── 工具/              # 分析工具、统计软件
└── 待分类/
```

## Wiki 概念页结构

```
wiki/concepts/
├── 论文/
├── 实验方法/
├── 数据集/
├── 理论/
└── 工具/
```
