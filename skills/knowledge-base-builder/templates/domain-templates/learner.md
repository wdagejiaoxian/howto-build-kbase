# 通用学习者模板

## 适用身份
学生、自学者、爱好者

## 预设分类

```
原始资料/
├── 课程笔记/          # 在线课程、培训笔记
├── 教程/              # 教程文章、操作指南
├── 项目实战/          # 实战项目、练习
├── 资源/              # 书单、工具、网站
└── 待分类/
```

## Wiki 概念页结构

```
wiki/concepts/
├── 课程笔记/
├── 教程/
├── 项目实战/
└── 资源/
```

## AGENTS.md 领域分类表参考

| 领域 | 概念页路径 | 包含内容 |
|------|----------|---------|
| 课程笔记 | wiki/concepts/课程笔记/ | 课程要点、知识点总结 |
| 教程 | wiki/concepts/教程/ | 操作步骤、最佳实践 |
| 项目实战 | wiki/concepts/项目实战/ | 项目经验、踩坑记录 |
| 资源 | wiki/concepts/资源/ | 推荐书单、工具、网站 |
