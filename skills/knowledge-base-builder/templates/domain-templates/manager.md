# 业务管理者模板

## 适用身份
产品经理、运营、市场营销人员

## 预设分类

```
原始资料/
├── 行业报告/          # 行业分析、市场趋势
├── 竞品分析/          # 竞争对手产品分析
├── 用户调研/          # 用户画像、反馈、需求
├── 方法论/            # 产品方法论、运营策略
├── 案例/              # 成功案例、失败教训
└── 待分类/
```

## Wiki 概念页结构

```
wiki/concepts/
├── 行业报告/
├── 竞品分析/
├── 用户调研/
├── 方法论/
└── 案例/
```
