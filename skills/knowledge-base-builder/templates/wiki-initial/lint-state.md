## 使用说明

| 命令 | 效果 |
|------|------|
| "运行 lint" | 从「待检查页面」列表中依次检查 |
| "继续 lint" | 从上次中断处继续检查 |
| "重新检查 xxx 分类" | 将该分类所有页面移回待检查 |
| "重新检查所有页面" | 清空已检查记录，全量重新检查 |
| "查看 lint 进度" | 汇报当前检查状态 |
| "清除 lint 记录" | 清空历史问题记录 |
